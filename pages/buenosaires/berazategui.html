Firma de convenio entre Berazategui y Agencia Nacional de Discapacidad

El lunes 13 de septiembre se llevó a cabo la firma de un convenio con el fin de destinar recursos del Fondo Nacional para la integración de las personas con discapacidad para adquirir elementos para el banco de ayudas técnicas para la municipalidad de Berazategui. .
